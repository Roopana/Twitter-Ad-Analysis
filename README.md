# Twitter-Ad-Analysis

## Objective
Identify tweets that refer to commercials played during super bowl 2020 game. 

## Methods used
### Bag of Words
- Glove
- Word2Vec

### Topic Modelling
Topic Modelling is an unsupervised technique to identify topics in a given text corpus based on joint distribution of words. [Gibbs Sampling algorithm for the Dirichlet Multinomial Mixture model](http://dbgroup.cs.tsinghua.edu.cn/wangjy/papers/KDD14-GSDMM.pdf) for short text clustering (GSDMM) is implemented in this project
